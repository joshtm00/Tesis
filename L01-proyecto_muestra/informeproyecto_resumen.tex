\begin{center}\huge{\textbf{Resumen}}\end{center}

%--------------------------------------------------------------------
\noindent

En el presente proyecto se plantea el desarrollo e implementación de programa de aplicación
para la realización de cortes vectoriales utilizando la cortadora láser. "Laser Spectrum H-Series 20x12 5th Gen".

Para el desarrollo de dicha aplicación se plantea un diseño modular donde diversos sub programas combinados ofrecen distintas funcionalidades, entre ellas se incluyen la extracción de información  gráfica a partir de un archivo PDF y la interpretación de dicha información para la generación de comandos de corte que puedan ser utilizados por la cortadora láser.

Estos diseños fueron implementados en el lenguaje de programación Python, donde se utilizó el paradigma de programación orientada a objetos para lograr la modularidad requerida por el diseño. 

Finalmente en el documento se presentan diversas pruebas de cortes realizados con el programa desarrollado y se plantean diversas recomendaciones para la el desarrollo futuro de la aplicación.
